## What BMI Tool Does

BMI Tool is a compact web application for estimating body mass index from basic height and weight information. Since it runs in the browser, you can open it immediately without installing a desktop program.

It is intended for fast, practical health checks when you want a BMI number and its category right away. The live-updating interface and support for multiple unit systems make it easy to compare readings in either Standard (US) or Metric form.

- Computes BMI from height and weight entries
- Works with both Standard (US) and Metric units

- Uses color-based output for easier interpretation
